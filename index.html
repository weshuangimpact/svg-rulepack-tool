SVG 規則產生器

rosbridge v2.1 · 解析 JSON → 逐參數指定規則(op) → 產生 Core config words → 匯出規則包
此工具只負責「把指令參數轉成 Core 規則字」。數值合理性由工程師判斷，裁決由燒錄後的硬體執行。
1 上傳 / 貼上指令封包（每行一則 JSON）
2 解析結果 → 逐參數指定規則
系統解析出每個 id（topic/service/action）與其參數。請為每個參數指定 op 與門檻值： LE≤上限　GE≥下限　EQ=精確值　ANY不限。門檻預設帶入觀測值(×1000定點)，可改。
3 產生 Core config words
依上表產生 32-bit config 字：[31:28]op [27:25]field [24:16]id [15:0]thresh。未指定規則的 v 槽自動填 ANY(不限)。
